# Suggested Commands

## Build and Run
- **Build**: `go build ./...`
- **Run**: `go run cmd/main.go`
- **Test all**: `go test ./...`
- **Test single**: `go test -run TestFunctionName ./path/to/package`
- **Test verbose**: `go test -v ./...`

## Code Quality
- **Lint**: `golangci-lint run` (if installed)
- **Format**: `go fmt ./...`

## Development Utils
- **Git**: Standard git commands (status, add, commit, push)
- **File operations**: `ls`, `cd`, `grep`, `find` (Linux system)
